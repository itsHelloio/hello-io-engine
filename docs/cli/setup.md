# `hello-io setup`

Initialize `~/.hello-io/hello-io.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
hello-io setup
hello-io setup --workspace ~/.hello-io/workspace
```

To run the wizard via setup:

```bash
hello-io setup --wizard
```
